# Polynomial Regression Visualization
PolyReg helps with visualization of how a polynomial regression model is trained using TensorFlow.

## Development Tools:
* [Git](http://git-scm.com/)
* [TensorFlow.js](https://js.tensorflow.org/)

## Give it a try:
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
